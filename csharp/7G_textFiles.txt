##Text Files

#Read Text

<code>
using System;
using System.IO;

using(StreamReader reader = new StreamReader(filename))
{
	string line = null;
	while((line = reader.ReadLine()) != null)
	{
	}
}
</code>

#Write Text

<code>
using System;
using System.IO;

using(StreamWriter writer = new StreamWriter(filename))
{
	writer.Write("X");
	writer.WriteLine("Y");
}
</code>

Redirect Console.WriteLine to a text file
<code>
using(var fileStream = new FileStream("name.txt", FileMode.Create)) {
	var streamWriter = new StreamWriter(fileStream);
	streamWriter.AutoFlush = true;
	Console.SetOut(streamWriter);
	
	Widget.DisplayReport(); //operation that displays to Console.WriteLine will now be saved to a file
}
</code>

#Write Text: Formatting

<code>
Console.Writeline($"{varA, 10} {varb, 20}");
</code>
This gives a 10-char-wide column for displaying varA, and a 20-char-wide column for varB
Positive column widths will right-align, and negative ones will left-align

#Operating System Separator

Windows vs Linux use different path or directory separators.
<code>
using System.IO;

var path = "folder" + Path.DirectorySeparatorChar + "filename.txt";
</code>

#Parts of File Name

get file extension
<code>
using System.IO;

string extension = Path.GetExtension("folder/file.txt");
</code>
