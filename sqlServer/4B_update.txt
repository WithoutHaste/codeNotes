##Update

Update records in a table:
<code>
UPDATE tableName SET columnA='Value', columnC=123 WHERE columnB >= 10;
</code>

#With Joins

<code>
UPDATE tableNameA SET columnA='Value', columnC=123
FROM tableNameA
INNER JOIN tableNameB on tableNameA.FkId = tableNameB.Id
WHERE tableNameA.columnB >= 10
AND tableNameB.columnC is not null;
</code>

#CASE

<code>
UPDATE tableName SET columnA = (CASE WHEN columnA=0 THEN 1 ELSE 0 END);
</code>
